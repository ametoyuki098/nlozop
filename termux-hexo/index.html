termux-hexo

发表于2025-08-23|更新于2025-10-28
|总字数:336|阅读时长:1分钟|浏览量:

介绍：此博文记录在安卓手机（平板）termux中安装配置hexo博客的过程，主题使用以及远程部署。

准备工作：

开始之前，请确保已准备好以下内容：

  • ✅ 安装termux 在F-Droid商店，或GitHub仓库中安装 （也可以安装修改版的zerotermux它更适合操作）
  • 🔲 一个域名 （可选）
  • ✅ GitHub账号
  • 🔲 Cloudflare （可选）

安装必要环境

  • 更新Termux的包信息和版本

在这个过程中遇到确认请求一律回车。可能有速度慢等网络问题，可以更换镜像源。

  • 在上个步骤完成后，执行以下命令安装nodejs

npm安装hexo

npm install hexo-cli -g

初始化hexo

  1. 使用mkdir youname创建一个文件夹
  2. cd youname 进入此文件夹
  3. 在后续之前执行hexo -v查看是否安装成功
  4. 再执行hexo init进行初始化创建
  5. npm install安装所需依赖

你也可以依次执行官方的安装操作

blog只是一个示例，你可以自定义成其他。如demobolg myblog…..

生成启动本地预览

hexo generate生成静态文件
hexo server开启本地预览
执行完成后会返回一个预览地址http://localhost:4000/在浏览器打开，就能看见博客。
